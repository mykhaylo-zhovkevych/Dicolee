# Dicolee

In this project I want to create a locally running webapp with an API written in TS between the frontend and PostgreSQL.

## Features
- The data will be stored locally, and thanks to the API the frontend can stay relatively simple.
- For data storage, I will use PostgreSQL because I want to learn how to work with it and because it fits well for a dictionary application with structured and expandable data.
- I want to make a flexible data schema so that later I can change or add new fields to the existing ones. The data schema should also be relatively generic so other people can use it for personal purposes.
